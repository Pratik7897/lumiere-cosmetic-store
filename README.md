## 🌸 Features

### 🛍️ Customer-Facing
- **Product Catalog** — 16+ products across 7 categories (Skincare, Lips, Face, Eyes, Nails, Fragrance)
- **Search & Filter** — Real-time search by name, brand, or category
- **Sort Options** — By price, rating, name, or featured
- **Product Details** — Full descriptions, ratings, reviews, stock info
- **Shopping Cart** — Add/remove items, quantity controls, order summary
- **Checkout** — Delivery form with COD, UPI, and Card payment simulation
- **Order History** — View past orders with status tracking & reorder
- **Reviews & Ratings** — Submit and browse product reviews with star ratings

### ⚙️ Admin Panel
- **Dashboard** — Revenue, orders, products, and low-stock overview
- **Product Management** — View all products with status indicators
- **Stock Control** — Update inventory levels in real-time
- **Add Products** — Add new items to the catalogue
- **Order Management** — View and update order statuses

### 🎨 Design
- Premium dark-plum & rose color palette
- Cormorant Garamond + DM Sans typography
- Glassmorphism topbar with backdrop blur
- Smooth hover animations & micro-interactions
- Fully responsive (mobile, tablet, desktop)
- Custom scrollbars

## 🏃 Getting Started

```bash

# Open in browser
open index.html
```
